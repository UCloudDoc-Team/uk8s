# 集群版本与规格说明

## 版本支持

uk8s 目前支持 kubernetes 1.19 ~ 1.34,其中1.30以上为推荐版本。如果你的集群在1.19到1.25之间,并且使用了 nginx ingress,建议尽快升级,因为官方已在2026年3月停止维护该组件。

## 节点规格

单个集群最多支持1000个节点,每个节点默认可以调度110个pod。如果需要更高密度,可以通过修改 kubelet 的max-pods参数,最高提升到256个。

## 证书与密钥

tls 证书需要用base64编码后填入 secret。例如把证书文件转成base64:

```shell
cat tls.crt | base64 -w0
```

## 带宽说明

外网带宽计费示例:10Mbps包月,或者300Mbps按时付费。共享带宽模式下带宽上线为2000Mbps。EIP 的p95带宽利用率超过80%时会触发告警。
